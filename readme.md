**Overview**



The most vital resource for life, aside from land and water, is air.As a result, air pollution is still one of China's most enduring environmental problems, and tracking changes in the Air Quality Index (AQI) is crucial to comprehending its effects on public health and urban sustainability.

As a result of numerous technological developments, a large amount of ambient air quality data is produced and utilized to determine the air quality in various locations.

## How to Run the Streamlit App

### Prerequisites
- Python 3.7 or higher
- Required packages (install from requirements.txt)

### Installation
1. Clone or download this repository
2. Navigate to the project directory:
   ```bash
   cd CMP7005
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running the App
Run the following command in your terminal:
```bash
streamlit run app.py
```

The app will start and open in your default web browser at `http://localhost:8501`. You can then navigate through the different pages to explore the air quality analysis and visualizations.
